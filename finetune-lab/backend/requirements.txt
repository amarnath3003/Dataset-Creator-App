# --- ML / GPU stack ------------------------------------------------------
# WARNING: `pip install -r requirements.txt` on its own gives you the CPU-only
# torch wheel from PyPI (silently — no error, `torch.cuda.is_available()` just
# returns False). Recommended install order on a GPU box:
#
#   1) pip install -r requirements.txt          # resolves all pins (may pull CPU torch)
#   2) python -c "import torch; print(torch.__version__)"   # note the version, e.g. 2.10.0
#   3) pip install --index-url https://download.pytorch.org/whl/cu126 \
#          torch==<that-version> torchvision --force-reinstall --no-deps
#   4) python -c "import torch; print(torch.cuda.is_available(), torch.__version__)"
#      # expect: True  2.x.x+cu126
#
# Pick the CUDA tag with the matching torch version — cu126 carries 2.10.0/2.11/2.12,
# cu124 tops out at 2.6.0 (see https://download.pytorch.org/whl/). A newer driver
# runs older cu wheels fine (forward compatible).
#
# Verified working combo (RTX 3070, driver 591.59): torch 2.10.0+cu126,
# torchvision 0.25.0, bitsandbytes 0.49.2, unsloth 2026.6.9.
